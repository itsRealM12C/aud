AU Extractor

Decodes raw GSM 06.10 payloads using the real ffmpeg decoder in-browser (ffmpeg -f gsm -ar <rate> -ac 1) — no hand-written codec, no guessing.

Header dump (first 64 bytes)

GSM frames are 33 bytes and stateful — each frame decodes using the previous frame's filter state, so a wrong header offset corrupts every frame downstream. If decode sounds like noise, try the alignment scanner before assuming the codec is wrong.
